# Retail Sales Performance Dashboard

## Project Overview

This project analyzes retail sales data to evaluate business performance, identify sales trends, and generate insights through data cleaning and interactive dashboard visualization.

The project workflow includes preprocessing raw data using Python and presenting insights through an interactive dashboard built with Looker Studio.

---

## Objectives

* Clean and prepare raw retail sales data
* Analyze sales performance and trends
* Build an interactive dashboard for business insights
* Support data-driven decision making

---

## Dataset

Source: Kaggle

This project uses a public retail sales dataset obtained from Kaggle for educational and portfolio purposes.

Dataset features include:

* Transaction Date
* Product Category
* Sales
* Quantity
* Revenue
* Customer Information
* Region
